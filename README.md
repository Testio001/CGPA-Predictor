# 🎓 Student CGPA Prediction and Academic Performance Analysis System

A Java-based console application that predicts student CGPA and 
analyzes academic performance. Built for Elizade University (COS 202 Project).

---

## 📌 Features

- Student registration (ID, Name, Department, Level)
- Course registration with grade entry
- GPA calculation per semester
- CGPA prediction based on semester history
- Academic standing classification (First Class, 2:1, 2:2, etc.)
- Best and weakest course detection
- Pass/fail rate summary
